# About

It's an travel app using python and Django.The users can select their destination and activities.

# App modules

* Home
* Admin
* Register
* Log in
* Log out

# Tech Features

* Open Source
* Languages Used:Python,HTML,CSS
* Framework Used:Django,Bootstrap
* Database -PostgreSQL
* IDE Used:Visual Studio Code
* Git integrations
